# Interactive mode

| Shortcut | Action |
| :--- | :--- |
| `?` | Toggle the keyboard shortcut help panel. Requires fullscreen rendering(/en/fullscreen) |
| `!` | Supports history-based autocomplete: type a partial command and press Tab to complete from previous `!` commands |

Press `Ctrl+O` to open transcript mode.

In transcript mode, press `/` to search and `n` or `N` to move through matches.

Run `/keybindings` to open your configuration file.
